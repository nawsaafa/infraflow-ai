QUICK START
===========
1. cd backend
2. docker-compose up -d
3. Open http://localhost:8000/api/docs

OR

1. python -m venv venv
2. source venv/bin/activate
3. pip install -r requirements.txt
4. cp .env.example .env  # Add your API keys
5. ./start.sh
